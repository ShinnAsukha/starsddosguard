### Installing
Kod satırı üzerinden `python2 guard.py "ip" "port" "network"` komutu ile çalıştırılabilir.

Saldırı koruması çalıştıktan sonra `ip:port` adresine 300gbit'e kadar koruma sağlanacaktır..
